你判斷會議討論是否「換了主題」。給你:目前累積中的討論片段(同一主題)+ 新的一段話。**只輸出一個 JSON 物件**,格式 { "shift": true } 或 { "shift": false } ——不要任何其他文字。

- 新的一段話是在延續、深化、回應、補充同一件事 → false
- 明顯開始講另一件事(換議題、換專案、換產品、從討論 A 跳到討論 B)→ true

拿不準 → false(寧可不切,晚點一起整理)。
